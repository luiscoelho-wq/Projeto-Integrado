# Requisitos Funcionais (RF)

RF01 — Exibição da tela inicial
O sistema deve apresentar uma tela inicial permitindo acesso às funcionalidades principais do aplicativo.

RF02 — Visualização do Dashboard
O sistema deve exibir um dashboard contendo indicadores estratégicos simulados e informações gerais do sistema.

RF03 — Navegação entre telas
O sistema deve permitir a navegação entre as telas principais por meio de componentes de interface.

RF04 — Pesquisa informativa
O sistema deve permitir que o usuário realize pesquisas utilizando uma barra de busca para consulta de orientações contábeis simuladas.

RF05 — Exibição de resultados da pesquisa
O sistema deve apresentar informações explicativas após a realização de uma pesquisa pelo usuário.

RF06 — Visualização de lista de clientes simulados
O sistema deve permitir a visualização de clientes fictícios cadastrados para fins de demonstração.

RF07 — Consulta de detalhes do cliente
O sistema deve exibir informações estratégicas simuladas ao selecionar um cliente da lista.

RF08 — Simulação básica de cenário tributário
O sistema deve permitir a alteração de parâmetros simulados para visualização de cenários tributários educacionais.

RF09 — Geração de resumo para reunião
O sistema deve permitir a visualização de um resumo estratégico baseado em dados simulados.

RF10 — Acesso ao modo estratégico
O sistema deve apresentar uma visão geral simulada da carteira de clientes.

RF11 — Atualização visual das informações
O sistema deve atualizar os dados exibidos conforme interações realizadas pelo usuário.

RF12 — Organização visual das interfaces
O sistema deve apresentar as informações organizadas em componentes visuais estruturados conforme o protótipo definido.
